Field Aliasing

Sauce assumes that all columns are named as they will appear in the Databse.
This can be changed on a per property basis {% highlight csharp %} using System; using System.Collections.Generic; using System.Linq; using System.Text; using DataAccess.Core.Attributes; namespace SauceExample { public class MyObject { public int id { get; set; } [DataField(FieldName="AnotherField")] public string Name { get; set; } } } {% endhighlight %}
