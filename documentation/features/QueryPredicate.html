Query Predicates

With sauce you can speicfy a query predicate, this is automatically added to all linq queries for a particular type
You can specify more than one per object {% highlight csharp %} public class TestItemWithPredicate { public int ID { get; set; } public string Name { get; set; } public bool IsDeleted { get; set; } [QueryPredicate] private static IQueryable QueryPredicate(IQueryable query) { return query.Where(r => r.IsDeleted == false); } } {% endhighlight %}
