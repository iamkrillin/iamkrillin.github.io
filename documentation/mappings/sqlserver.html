Sql Server CLR DB Type Mappings

  • DateTimeOffset -> DateTimeOffset(7)
  • String -> varchar(200)
  • Int32 -> int
  • int64 -> bigint
  • single, double -> real
  • byte[] -> varvinary(MAX)
  • Boolean -> bit
  • DateTime -> DateTime
  • TimeSpan -> time(7)
