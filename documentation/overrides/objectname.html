Custom Column Loading

Sauce will assume that all object are named as such $(CLR_Object_Name)s
This can be changed on a per object basis {% highlight csharp %} using System; using System.Collections.Generic; using System.Linq; using System.Text; using DataAccess.Core.Attributes; namespace SauceExample { [TableName(TableName="SomeOtherTableName")] public class MyObject { public int id { get; set; } public string Name { get; set; } } } {% endhighlight %}
