Default Values

Sauce will not attempt to insert a value for a null property.
However, you can specify a defualt value for a null column (db specific)
Note: this is not enforced by sauce (maybe later) {% highlight csharp %} using System; using System.Collections.Generic; using System.Linq; using System.Text; using DataAccess.Core.Attributes; namespace SauceExample { public class MyObject { public int id { get; set; } [DataField(DefaultValue="Bob")] public string Name { get; set; } } } {% endhighlight %}
