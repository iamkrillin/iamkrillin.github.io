Column Loading

By default all properties on an object will load from the DB all the time. You can change this behavior on a per column basis.
This can be particully useful for blobs and the like. {% highlight csharp %} using System; using System.Collections.Generic; using System.Linq; using System.Text; using DataAccess.Core.Attributes; namespace SauceExample { public class MyObject { public int id { get; set; } [DataField(LoadField=false)] public string Name { get; set; } } } {% endhighlight %}
